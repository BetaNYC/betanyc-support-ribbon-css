About the ribbon

This is a pure CSS implementation of the famous Fork me on GitHub CSS ribbon to show support for BetaNYC.

Usage

To use it on your site, just view source and copy the two marked sections (one the CSS, the other the HTML tag). You'll also need to grab the CSS files here.

Feel free to link to the following BetaNYC sites:

  • BetaNYC Landing Page
  • BetaNYC Donation Page on Code for America
  • BetaNYC Developers List
  • Alternatively, fork this civic project!

Using with Bower

You can install the CSS files using Bower. 

bower install github-fork-ribbon-css
